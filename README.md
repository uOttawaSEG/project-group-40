# Programming Standards
Visibility should always be explicitly declared.
Type may be implicitly declared if it's clear what it is (ie: var x = new Student())
Don't be afraid of whitespace, please put a space for variable assignments, parameters, etc.


## Naming conventions
* Private variables should always start with an underscore (`_`) and follow camelCase naming conventions.
* Parameters should also be camelCase.
* Public variables should always be in PascalCase
* Classes should always be in PascalCase

## Classes
* Classes should be organized in the following manner:
* Private fields
* Protected fields
* Public fields

* Constructor

* Private methods
* Protected methods
* Public methods

## Enums
* Enums should be capitalized like classes
* All enum values should be in SCREAMING_SNAKE_CASE
